FAQ

1. Quem é responsável pelo projeto?

Somos um grupo de apaixonados por tecnologia. O projeto que começou como iniciativa da Data Science Brigade, braço de data science da CODELAND, cresceu e hoje tomou vida própria, contando também com diversos profissionais de fora da empresa. Todo código é aberto, contamos com um corpo de centenas de voluntários: colaboradores do mundo inteiro, incluindo jornalistas e investigadores.

2. Onde vocês estão?

Nosso time é espalhado por cidades e países diferentes. Usamos dos benefícios da tecnologia para conseguirmos fazer reuniões, discutir o projeto e dar continuidade do trabalho de forma remota, sem precisarmos de um espaço físico.

3. O que é open source?

Open source é um termo em inglês utilizado para distribuição de código de forma aberta, onde todas as pessoas interessadas em contribuir para a evolução do projeto podem, de forma livre, agregar suas ideias.

4. Quais dados utilizamos?

Utilizamos dados públicos — seja dados tornados públicos pela Lei de Acesso a Informação, seja dados privados disponibilizados publicamente por empresas como Google, Foursquare, Yelp etc. Na esfera pública conseguimos dados junto à Câmara dos Deputados, Receita Federal, Portal da Transparência, dados.gov.br etc.

5. O que é a cota para exercício de atividade parlamentar?

A Cota para Exercício de Atividade Parlamentar (CEAP) é um valor mensal de até R$ 45 mil que cada deputado tem direito a ressarcimento por gastos que não cabem licitação. Um almoço ou pagamento de táxi, por exemplo.

6. Como as denúncias são feitas?

As denúncias são feitas diretamente à Câmara dos Deputados através de um painel de casos com a maior probabilidade de ilegalidade encontrados pela Rosie. Criamos algumas ferramentas internas que pretendemos refinar para poder oferecê-las publicamente em breve.

7. Para onde posso enviar dúvidas/elogios/sugestões?

Você pode entrar em contato através da página da operação no Facebook ou falar com o Pedro, nosso mestre da comunicação, por email. Se for da área técnica, pode falar com a gente direto pelo GitHub também.

8. Qual o canal técnico de comunicação do projeto?

Atualmente conversamos na língua inglesa, em um grupo aberto do Telegram. Você é muito bem vindo(a) para se juntar a ele.

9. Por que a comunicação técnica é feita em inglês?

Contamos com uma comunidade criada em volta do projeto que vai além do Brasil - pessoas de outros países interessadas em contribuir. Dentro do repositório do Github e em nosso canal do Telegram, optamos por usar a língua inglesa para que possamos incluir as opiniões dessas pessoas na discussão. Além disso, várias pessoas de outros países demonstraram interesse em nosso código desde o dia zero. Mantendo o código e toda documentação técnica em inglês, fazemos com que facilmente o nosso esforço possa ser aproveitado em outros países do mundo.

10. O que é Machine Learning?

Para falar sobre Machine Learning, selecionamos um vídeo que sumariza tudo que precisamos saber para entender sobre o assunto: O que é machine learning?

11. O que é inteligência artificial?

Escrevemos um post sobre a diferença Entre Inteligência Artificial, Machine Learning e Deep Learning — acho que ele pode te ajudar.

12. Por que estamos usando python?

Usamos python por que ele nos fornece a grande maioria de bibliotecas disponíveis para análise de dados e aprendizagem de máquina.

13. O que é o Jarbas?

O Jarbas é uma ferramenta criada por nós para facilitar a visualização dos dados que estão sendo minerados por nosso Robô. Essencial para o trabalho investigativo. Ele surgiu como uma ferramenta interna mas aos poucos vamos melhorar a interface dele para que ele se torne uma ferramenta de utilidade pública.

14. O que é a Rosie?

A Rosie é a nossa Robô que foi programada para identificar usos ilegais da verba pública, começando pela CEAP. Ela julga cada um dos reembolsos solicitados pelos nossos deputados, nos dizendo quais são os motivos que fazem cada um poderem ser ilegais.

15. Como o projeto nasceu?

A primeira fase do projeto, que durou dois meses, foi realizada através de uma campanha de financiamento coletivo. Mais de 1296 nos ajudaram a tornar o projeto realidade.

16. Como posso contribuir com o projeto?

Você pode contribuir para o projeto com código, ajudando a manter a equipe, com ideias, com retuítes e compartilhamentos. Para entender como começar a contribuir com código, leia este post. Para contribuir diretamente para a continuação do projeto, você pode acessar a nossa página no apoia.se.

17. O que o nome Serenata de Amor significa?

Os motivos são três:
1 - Nos inspiramos diretamente no famoso Caso Toblerone - um famoso acontecimento onde uma política na Suíça renunciou o seu cargo após ser flagrada com um simples Toblerone na fatura de seu cartão corporativo. Queremos fazer isso: encontrar corrupção em pequenos gastos, mas em volume grande.
2 - Parece nome de operação da Polícia Federal, e isso é muito legal.
3 - É a nossa Serenata de Amor para o Brasil.

18. Vocês estão protegendo os deputados não divulgando o nome dos suspeitos?

Não, estamos protegendo a nós e a nossa comunidade. Para minimizar a chance de processo de crime contra a honra só divulgamos nomes quando alguma instância oficial corrobora nossas suspeitas. Mas os dados são abertos e nossas ferramentas também, fique à vontade para fazer uma auditoria você mesmo!
